Fluxograma de Controle de Estoque

Passo 1: Iniciar

Forma: Oval

Indica o início do processo.

Passo 2: Solicitar Saldo Inicial

Forma: Retângulo

O sistema solicita o saldo inicial da peça.

Passo 3: Loop para Entrada de Dados

Forma: Losango

Verifica se o usuário deseja continuar inserindo dados.

Passo 4: Entrada do Tipo de Operação

Forma: Retângulo

Solicita ao usuário o tipo de operação (1=compra, 2=venda).

Passo 5: Entrada da Quantidade

Forma: Retângulo

Solicita ao usuário a quantidade de peças.

Passo 6: Verificação do Tipo de Operação

Forma: Losango

Verifica se a operação é uma compra.

Passo 7: Operação de Compra

Forma: Retângulo

Adiciona a quantidade ao saldo se for uma compra.

Passo 8: Verificação de Saldo para Venda

Forma: Losango

Verifica se a quantidade para venda é menor ou igual ao saldo disponível.

Passo 9: Operação de Venda

Forma: Retângulo

Subtrai a quantidade do saldo se a venda for possível.

Passo 10: Mensagem de Saldo Insuficiente

Forma: Retângulo

Exibe uma mensagem se não houver saldo suficiente.

Passo 11: Exibir Saldo Atualizado

Forma: Retângulo

Mostra o saldo atualizado após cada operação.

Passo 12: Encerrar o Sistema

Forma: Oval

Indica o encerramento do processo.
